# House of UGC — UI Kit

A high-fidelity interactive prototype of the House of UGC agency website.

## Screens
- `index.html` — Full agency website (hero, services, stats, process, testimonials, CTA, footer)

## Design Notes
- Dark theme: #0A0A0A background
- All sections use real design system tokens from `../../styles.css`
- Scroll-reveal animations on all major sections
- Responsive down to ~768px
- Interactive: nav CTAs scroll to sections, email CTA form is styled

## Usage
Open `index.html` directly in a browser to see the full interactive prototype.
